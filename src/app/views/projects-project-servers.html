On Premise Server
You install the server yourself on your hardware


1
Prerequisites
Please, Resolve all prerequisites to run

For deploy your new Homer server, you need basic environments. For example on Microsoft Azure - Own Virtual Machine. Ubuntu Linux v.16+ with latest update is recommended.

Before - check that you have open this Ports

On Microsoft Azure or AWS, you have to do that manually!

  • 1881 (MQTT)
  • 1883 (TEST MQTT)
  • 3000 (API)
  • 3433 (API)
  • 8503 (GRID Apps)
  • 8501 (Terminal WebView)
  • 8505 (Terminal HW logs)
  • 10233 (Reserve Mongo API)
2
Download
Download latest recommended release for selected platform
  • Linux
  • Mac OS X
  • Windows 7, 8, 10
3
Install with Terminal Commands
For more details look at Docu

Find your file in your directory and use this commands:

  • chmod +x homer-server-linux
  • sudo ./homer-server-linux &>/dev/null &
4
Set CName Record for your IP Address
Fallow Installation Tutorial

For Secure Configuration with Certificates Set CName Record on your Domain to local IP address of your server

And Wait a few Minutes. Try to use your new domain configuration mycnamename.my_domain.com:3000.

If you see configuration page, you can continue to next step.

5
Create New Server Record on Byzance
Fallow Installation Tutorial

And don't forget to use your mycnamename.my_domain.com instead IP Address!

Server will automatically configure your server and also install domain certificates.
